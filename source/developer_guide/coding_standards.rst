コーディング規約
================

PHP コード
--------

-  インデント: スペース 4 文字
-  改行: Unix => ``\n``
-  文字コード: UTF-8
-  開始タグはテンプレートの ``<?php`` か ``<?=`` のみを使用し,  ``<?`` は **決して**使用しない
-  常にメソッドとクラスのプロパティにPHPdoc のコメントを付ける
-  コーディングスタイル: `PSR-1 <http://www.php-fig.org/psr/psr-1/>`__ と `PSR-2 <http://www.php-fig.org/psr/psr-2/>`__ に従う

JavaScript コード
---------------

-  インデント: スペース 4 文字
-  改行: Unix => ``\n``

CSS コード
--------

-  インデント: スペース 4 文字
-  改行: Unix => ``\n``
